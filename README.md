# react-express-mongoose-cms

公司的一个小网站用react来练习了下。
目前把最复杂菜单列表(2级而已...)前后台搞定了,其他是静态页面。
把data文件夹里面的json文件分别再mongodb里建2个文档(news, navTypes)，然后插入下数据。

* npm install
* bower install
* gulp
* npm run start
* go to [Web](http://127.0.0.1:3000/web)
* and the [Admin Page](http://127.0.0.1:3000/admin)
